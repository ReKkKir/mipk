Информация - это сигналы, получаемые человеком с помощью органов чувств и различнх приборов

Информацию можно классифицировать:
- по способу восприятия: на визуальную, аудиальную, тактильную, обонятельную и вкусовую

Вид информации Органы чувств Пример
Визуальная Зрение С помощью зрения мы рассматриваем картины, читаем книги
Аудиальная Слух С помощью слуха воспринимаем музыку, звук машин
Тактильная Осязание С помощью кожи чувствуем холодную стену, гладкость поверхности стола
Вкусовая Вкус С помощью рецепторов на языке мы узнаём, какое мороженное вкусное, например
Обонятельная Обоняние С помощью обоняния мы ощущаем запахи

- по способу представления информация бывает: текстовая, числовая, графическая, цифровая, мультимедийная
